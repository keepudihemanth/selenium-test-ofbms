### selenium-test
Selenium testing on a online flight booking system.
Building a scripting for automated testing on a online flight booking system. 

---
### Note
This is only java script for testing
---
Ultimate goal is to check how good the software is and give reports on the automated results.

### How to Run
Clone this repo
Install maven dependencies 
use command: mvn test -Dtest=AdminTest // for admin test
use command: mvn test -Dtest=LoginTest // for login test
